## 主题系统说明

### 设计原则

- 所有颜色通过语义化 **Token**（`SolidColorBrush`）定义在 `LightTheme.xaml` / `DarkTheme.xaml`
- 样式文件中通过 `DynamicResource` 引用 Token，运行时替换主题字典即可切换
- 各页面 Background 使用 `{DynamicResource PageBackgroundBrush}` 或 `PageBackgroundAltBrush`
- 设计差异较大的页面（CsvComparePage、DataValidationPage）保留完整本地样式，仅 Background 主题化

### 颜色 Token 体系

| 类别 | Token 示例 | 用途 |
|------|-----------|------|
| 品牌色 | `PrimaryBrush` `PrimaryDarkBrush` `PrimaryLightBrush` | 主按钮、选中态、强调 |
| 状态色 | `SuccessBrush` `ErrorBrush` `WarningBrush` `PurpleBrush` | 成功/错误/警告/特殊标记 |
| 背景色 | `PageBackgroundBrush` `PageBackgroundAltBrush` `CardBackgroundBrush` | 页面/卡片背景 |
| 文字色 | `TextPrimaryBrush` `TextDarkBrush` `TextSecondaryBrush` `TextMutedBrush` | 多级文字层次 |
| 输入控件 | `InputBackgroundBrush` `InputBorderBrush` `InputFocusBorderBrush` | 输入框背景/边框/聚焦 |
| DataGrid | `GridHeaderBackgroundBrush` `RowHoverBrush` `RowSelectedBrush` `GridCellBorderBrush` | 表格各部位 |
| 导航栏 | `NavBackgroundBrush` `NavActiveBrush` `NavHoverBrush` `NavTextBrush` | 侧边栏 |
| 下拉框 | `DropdownBackgroundBrush` `DropdownBorderBrush` `DropdownArrowBrush` | ComboBox 弹出 |

### 页面本地样式说明

部分页面因设计语言差异较大，保留了完整的本地样式：

| 页面 | 保留的本地样式 | 原因 |
|------|---------------|------|
| **CsvComparePage** | `Card`、`BtnTpl`/`PrimaryButton`/`SecondaryButton`/`GhostButton`/`SuccessButton`、`ResultDataGrid`/`ResultDataGridHeader`/`ResultDataGridCell`/`ResultDataGridRow`、`ModeRadio`/`KeyColumnCheckBox`/`FilterChip`/`PageSizeCombo` | 按钮模板（圆角 9、带边框）、DataGrid（无边框单元格、38px 行高）、交互控件均与全局差异显著 |
| **DataValidationPage** | `RoundedInputBox`、`DvHeader`/`DvCell`/`DgMappingRow`（42px 行高适配内嵌 ComboBox）、`DgIssueRow`、`InlineCb`、`ValidationOptionChip`、`StepCircle` 系列 | 步骤向导式 UI、DataGrid 内嵌 ComboBox 需要更高行高、圆角输入框 |
| **InvoicePrintPage** | `RefreshBtn`、`LabelText`/`SubLabel` | 刷新按钮有独特的浅蓝边框风格 |
| **AppMessageDialog** | `DialogButtonBaseStyle`/`PrimaryButtonStyle`/`SecondaryButtonStyle` | 对话框按钮（圆角 10、固定高度 32）与全局按钮定位不同 |

### 主题切换方法

```csharp
// 切换到深色主题
ThemeManager.SwitchTheme("Dark");

// 切换回浅色主题
ThemeManager.SwitchTheme("Light");

// 获取当前主题
var current = ThemeManager.CurrentTheme; // "Light" 或 "Dark"
```

## 本次整理修复的 Bug

| Bug | 原因 | 修复 |
|-----|------|------|
| CSV 对比：第二次选不同主键对比报错 | `BindCurrentPage` 中 `_currentPageRows.Clear()` 时 DataGrid 仍引用该列表，触发 `SelectionChanged` 访问已清空数据 | 在 `Clear()` 前先置空 `ItemsSource` |
